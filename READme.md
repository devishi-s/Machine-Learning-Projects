# Machine Learning Projects

This repository contains three machine learning projects completed as part of an academic assignment. Each project focuses on solving a real-world prediction problem using data preprocessing, exploratory data analysis (EDA), machine learning, and model evaluation.

## Project Overview

### Part A: Property Price Prediction

Developed regression models to predict property prices using housing-related features. The project includes data cleaning, exploratory data analysis, feature engineering, model development, and performance evaluation using regression metrics.

### Part B: Customer Churn Prediction

Built a classification model to predict whether a customer is likely to churn based on demographic, account, and service-related information. The project involves data preprocessing, exploratory data analysis, feature engineering, model training, and evaluation using classification metrics such as Accuracy, Precision, Recall, and F1-Score.

### Part C: Early Disease Detection

Developed a machine learning model to predict the presence of heart disease using patient health and demographic data. The project includes exploratory data analysis, data preprocessing, model development, performance evaluation, and analysis of the key factors influencing disease prediction.

## How to Run

1. Clone this repository.
2. Install the required libraries:

```bash
pip install -r requirements.txt
```

3. Open any notebook using Jupyter Notebook or Visual Studio Code.
4. Ensure the corresponding dataset is placed in the same folder as its notebook before running the cells.
